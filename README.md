# Data-Analyst-Securitisation-PowerBI

## Project Overview

This project analyzes an Auto Loan Securitisation portfolio using Microsoft Power BI.

The dashboard provides:

- Executive Dashboard
- DPD Analysis
- Dynamic Loss Analysis
- Static Pool (Vintage) Analysis
- Investor Reporting

---

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel

---

## KPIs

- Total Loans
- Outstanding Balance
- Default Rate
- Recovery Rate
- Pool Factor
- Expected Credit Loss
- Collection Efficiency

---

## Dashboards

### Executive Dashboard

Shows portfolio summary and loan distribution.

### DPD Dashboard

Shows delinquency analysis.

### Dynamic Loss Dashboard

Shows monthly losses and recoveries.

### Static Pool Dashboard

Shows vintage performance and pool factor.
